## 🚀 Tez Boshlash

### Talablar
- Node.js 16+ 
- npm yoki yarn

### O'rnatish

1. **Repository ni klon qiling:**
```bash
git clone https://github.com/Muhammaddiyor2010/Maslahatchi_ai.git
cd Maslahatchi_ai
```

2. **Dependencies ni o'rnating:**
```bash
npm install
```

3. **Development server ni ishga tushiring:**
```bash
npm run dev
```

4. **Browser da oching:**
```
http://localhost:3000
```
